# Roadmap

## v0.2.0
- Docker and Podman plugins (container inventory, image/volume diff)
- Plugin registry for sharing community plugins
- Scheduled snapshot creation (systemd timer integration)
- Anomaly detection in diff output

## v0.3.0
- Optional TUI for browsing snapshots and diffs
- Interactive diff viewer

## v1.0.0
- Stable snapshot schema (backward-compatible versioning)
- Full API documentation and plugin SDK stability guarantees
